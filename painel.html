02 / Hospedar online

Coloque no ar em 30 segundos

Escolha um dos serviços abaixo. Todos são gratuitos, sem precisar de cartão, e geram uma URL pública pronta pra usar no QR code.

Recomendado

Vercel

A própria plataforma que vocês já usam (contourline-hub.vercel.app). Sub-domínio próprio incluso.

  1. Acesse vercel.com/new
  2. Faça login com GitHub/Google
  3. Arraste o arquivo formulario.html
  4. Em Settings > Domains defina ex.: cbcd.contourline-hub.vercel.app
Mais rápido

Netlify Drop

Drag-and-drop puro: solta o arquivo e em 5 segundos você tem URL pública.

  1. Acesse app.netlify.com/drop
  2. Arraste formulario.html na área marcada
  3. Pronto. Copie a URL gerada (ex.: nome-aleatorio.netlify.app)
  4. (Opcional) Renomeie o site nas configurações
GitHub Pages

GitHub Pages

Se preferir versionar o arquivo num repositório.

  1. Crie um repo público no GitHub
  2. Suba formulario.html renomeado para index.html
  3. Em Settings > Pages, ative main / root
  4. URL final: usuario.github.io/repo

04 / Checklist final

Antes de plotar o novo balcão

  1. Hospede o formulario.html em uma das opções da etapa 02 e copie a URL pública.
  2. Teste a URL no celular, escaneando o QR gerado. Preencha um cadastro de teste com seu próprio nome ("TESTE CONTOURLINE") para validar o caminho completo.
  3. Confirme na planilha do Google Forms original que a resposta de teste apareceu. Confirmado? Os 4 campos novos continuam alimentando a mesma planilha.
  4. Baixe o QR em 2000px (etapa 03) e envie para a gráfica. Mantenha pelo menos 2cm de margem branca no entorno para garantir leitura.
  5. (Opcional) Mantenha o QR antigo válido por algumas semanas — o link do Google Forms continua ativo, então quem ainda escanear o balcão antigo não fica sem opção.
